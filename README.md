# madalina patru — academic website

Personal academic website hosted on GitHub Pages.

## How to deploy

1. Create a repository named `yourGitHubUsername.github.io` on [github.com/new](https://github.com/new)
2. Upload all files (index.html, cv.html, research.html, teaching.html, style.css)
3. Go to Settings → Pages → Deploy from branch **main** / **root**
4. Wait ~2 minutes → your site is live at `https://yourGitHubUsername.github.io`

## How to add your photo

1. Upload a file named `photo.jpg` to the repository
2. In `index.html`, uncomment the `<img>` line and delete the placeholder `<div>`

## How to add your CV PDF

1. Upload your CV as `cv.pdf` to the repository
2. The "Download PDF" button on the CV page already links to it
